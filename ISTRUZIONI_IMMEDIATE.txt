🆘 SE HAI ANCORA ERRORI:
────────────────────────────────────────────────────────────

Errore: "Port already in use"
────────────────────────────────────────────────────────────
Qualcosa sta usando le porte. Trova e termina:

# Trova cosa usa la porta 3000
netstat -ano | findstr :3000

# Termina il processo (sostituisci 1234 con il PID trovato)
taskkill /PID 1234 /F


Errore: "Cannot connect to Docker daemon"
────────────────────────────────────────────────────────────
Docker Desktop non è avviato:
1. Apri Docker Desktop
2. Aspetta che sia pronto (icona balena ferma)
3. Riprova


Errore durante il build
────────────────────────────────────────────────────────────
Pulisci e riparti:

# Ferma tutto
docker-compose down

# Pulisci cache Docker
docker system prune -f

# Ricostruisci
docker-compose up --build


🔍 VERIFICA MANUALE:
────────────────────────────────────────────────────────────

# Controlla che Docker sia attivo
docker --version

# Lista container in esecuzione
docker ps

# Visualizza logs
docker-compose logs -f
